# ReportAds / Relatório360

Protótipo web para upload de planilhas de campanhas e geração de relatórios por cliente.

## Funcionalidades atuais

- Tela de login demonstrativa.
- Perfil administrador e perfil cliente.
- Painel admin para gerenciar usuários e selecionar cliente antes do upload.
- Upload de CSV/XLSX no navegador via SheetJS.
- Filtros por cliente, canal, período e busca.
- KPIs de investimento, receita, conversões, CPA, ROAS e CTR.
- Gráfico simples por data.
- Tabela de performance por campanha.
- Resumo executivo e recomendações automáticas.
- Exportação/impressão do relatório.

## Acessos de demonstração

- Admin: `admin@relatorio360.com` / `admin123`
- Cliente: `aurora@cliente.com` / `cliente123`

## Observação

Esta versão é um protótipo estático. Para produção, autenticação, permissões, armazenamento de usuários e campanhas devem ser migrados para backend real com banco de dados.
